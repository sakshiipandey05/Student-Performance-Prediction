# Student Performance Prediction

This project uses Machine Learning to predict a student's final score based on:

- Study Hours
- Attendance Percentage
- Previous Score

The model is built using Scikit-Learn and deployed through an interactive Streamlit dashboard for real-time predictions.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Streamlit

## Features
- User-friendly dashboard
- Real-time score prediction
- Interactive input sliders
- Machine Learning based analysis
